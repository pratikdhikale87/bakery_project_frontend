# Rosy Oven Bakery UI

React + Tailwind frontend project for a bakery e-commerce site.

## Included pages

- Landing page with hero, categories, delivery flow, testimonials, and footer
- Products page with category filters, search, and sorting
- Product details page with 6-image gallery and auto-rotation every 5 seconds
- Cart and checkout flow
- Login/register page
- User dashboard for orders and tracking
- Admin dashboard for product creation and order management

## Run

```bash
npm install
npm run dev
```

## Notes

- Data is mocked and stored in `localStorage`
- Product images are generated SVG placeholders, so the demo works without external assets
